## 輸入
一行包含兩個整數，一個運算子的算式，格式如：  
$A + B$  
可能出現的運算子有+，-，*，以及/。  
除法不會出現負數。  

## 輸出
輸出運算結果。注意，老師要求好學生書寫算式要過程完整，字型整齊。因為小明是個好學生，所以他被要求按照嚴格的格式書寫！要把原本的題目謄寫一遍再加上答案喔。  

## 輸入限制
 - $-10^{18} \leq A, B \leq 10^{18}$

## 子任務
| 編號 | 分數 |    限制    |
| :---: | ---: | ---------- |
|  1  | 2 | 範例輸入輸出 |
|  2  | 18 | $A = 0$、$1 \leq B \leq 10^{4}$ |
|  3  | 30 | $0 \leq A, B \leq 10^{4}$ |
|  4  | 50 | 無額外限制 |

\clearpage

## 範例輸入
```
-42 * 9
```

## 範例輸出
```
42 * 9 = -378
```
